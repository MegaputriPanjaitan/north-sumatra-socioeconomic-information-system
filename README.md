# 📊 North Sumatra Socioeconomic Information System

An interactive **Power BI dashboard** for analyzing socioeconomic conditions across regencies/cities in **North Sumatra Province (2019–2023)** using official data from **Statistics Indonesia (BPS)**.

## 🎯 Project Objectives

- Analyze Human Development Index (HDI)
- Monitor Poverty
- Analyze Employment (TPT & TPAK)
- Explore Population Distribution
- Support data-driven decision making

## 📂 Dashboard Pages

- Executive Overview
- HDI Analysis
- Poverty Analysis
- Employment Analysis
- Demographic Analysis
- Executive Summary & Key Insights

## 🛠 Technologies

- Python (Pandas)
- Power BI
- Git & GitHub
